# Dynamic Pricing UI (Angular)

## Features
- Dynamic pricing table
- Signal-based state management
- JSON-driven UI
- Supports:
  - Flat pricing
  - Size-based pricing
  - Fixed & percentage charges

## Tech Stack
- Angular (latest)
- Signals (no RxJS)

## Run Locally
npm install
ng serve

## Notes
- Built using standalone components
- Focused on scalability and clean structure
